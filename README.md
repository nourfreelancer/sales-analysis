# Sales Analysis Automation

This project reads a CSV file containing sales data,
cleans the data, and calculates total quantity sold,
total sales amount, and top selling product using Python.

## Files
- sales.csv: Sample sales data
- sales_analysis.py: Python script for analysis
- sales_summary.csv: Generated summary report (output of the script)

## Output
- Total Quantity Sold
- Total Sales Amount
- Top Product
